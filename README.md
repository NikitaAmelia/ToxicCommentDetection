# 🛡️ Toxic Comment Detection System

A machine learning–based web application designed to automatically detect and classify toxic or harmful comments using Natural Language Processing (NLP).  
This project aims to promote safer online communication by identifying hate speech, offensive language, and toxic content in real time.

---

## 🚀 Project Overview

Online platforms increasingly face challenges related to toxic interactions and harmful communication.  
This project provides an AI-powered solution that analyzes user text input and predicts whether a comment contains toxic language.

The system integrates machine learning models, NLP preprocessing, and an interactive web interface to deliver real-time predictions.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/NikitaAmelia/ToxicCommentDetection.git
cd ToxicCommentDetection
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

**Windows**
```bash
venv\Scripts\activate
```

**Mac/Linux**
```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Run locally using:

```bash
python app.py
```

or if using Streamlit:

```bash
streamlit run app.py
```

Then open your browser at:

```
http://localhost:8501
```
